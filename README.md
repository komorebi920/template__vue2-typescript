# vue2-typescript-template

## 运行环境

1. `yarn^1.22.19`
2. `node^14.21.3`

## 浏览器兼容性

1. `ie` >= 9
2. `firefox` >= 52
3. `chrome` >= 76

## 模板功能

1. `vue + vue-router + vuex` 全家桶
2. `jest` 单元测试
3. `vue-i18n` 多语言
4. `element-ui` UI 组件库
    - 按需加载
    - 多语言
    - 自定义主题
5. Commit Message 校验
6. Prettier 格式规范
7. 同时支持网页应用打包与库模式打包
    - 库模式自动生成 demo 页面
    - 导出 Class 类文件
    - 获取当前 Git 分支、提交时间以及 Commit ID
